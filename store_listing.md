## English Description (Long — 4000 characters)

**Farketmez** ("Doesn't Matter") — stop arguing about where to go and start voting!

We've all been there: ten people, ten opinions, zero decisions. Farketmez turns group indecision into a quick, fun vote so you can actually get going.

**How it works**

1. Create a room — choose Food or Activity, share the code
2. Friends join — they enter the code on their phone
3. Host starts voting — nearby places are fetched automatically via GPS
4. Everyone votes — ❤️ for yes, ✕ for no
5. Results revealed — highest-voted place wins!

**Features**

🗺️ **Smart Place Discovery**
Powered by Google Places API. Restaurants, cafes, parks, museums and more are listed around your location, complete with star ratings, review counts, price level and open/closed status.

📸 **Place Photos**
Each venue card shows a real photo from Google Places, so you know what you're voting for before you arrive.

📍 **Map View**
Switch between list and map views, tap any pin to see details and vote right there.

⚡ **Live Updates**
Participant list and vote counts refresh automatically — no need to pull to refresh.

🏆 **Score System**
Earn bonus points for picking the winner, lose points for not voting. A lighthearted way to keep everyone engaged.

🔒 **Session Memory**
The app remembers your session. Close it, reopen it, and you're right back in the same room with the same nickname.

🔍 **Category Filters**
Filter places by pizza, burger, cafe, steakhouse, fast food, and more. The host can add extra categories mid-vote to widen the search.

🇹🇷 **Built for Turkey**
Turkish language throughout. Quick-pick city list for manual location entry. Search radius automatically expands if not enough places are found nearby.
